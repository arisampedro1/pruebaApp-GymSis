# Prueba App GymSis
Aplicación desarrollada en Python para gestionar rutinas de gimnasio de forma simple.
Permite organizar ejercicios por día, agregar series, repeticiones, peso y tiempos de descanso, además de editar o eliminar rutinas según sea necesario.

## Funcionalidades
- Crear rutinas por día
- Agregar ejercicios
- Editar series, repeticiones y peso
- Eliminar ejercicios
- Eliminar días completos
- Borrar toda la rutina
- Configurar descanso entre series
- Configurar descanso entre ejercicios
- Iniciar una rutina con temporizador

## Cómo ejecutar el proyecto

1. Clonar este repositorio
2. Abrir la carpeta del proyecto
3. Ejecutar el archivo principal con Python
